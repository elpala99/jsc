\section{Untyped Lambda Calculus}
We suppose an infinite set of variables called \mathcal{Var}. The, we define the terms of the lambda calculus the following way:
\begin{itemize}
  \item $$\forall x \in \mathcal{Var}$$, $x$ is a term.
\end{itemize}
